# 簡単な数学

```Python
print(1 + 2) #足し算
print(1 - 2) #引き算
print(1 * 2) #掛け算
print(2 ** 2)#べき乗
print(1 / 2) #割り算
print(1 // 2)#
print(1 % 2) #余り
```



Python2では整数割る整数は整数が結果として得られたが，Python3では整数割る整数は浮動小数点数が結果として得られる．
例えば1/2はPython2では0だが，Python3では0.5となる．
